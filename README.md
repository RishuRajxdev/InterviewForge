AI-powered interview preparation platform. Upload a resume and a target job description, and get a skill-gap analysis, tailored interview questions, and a prep plan generated by AI.

## Features

- Resume upload and parsing (PDF/DOCX)
- Skill gap analysis against a target job description
- AI-generated technical and behavioral interview questions
- Personalized interview prep flow
- JWT authentication with token blacklisting

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React (Vite), SCSS |
| Backend | Node.js, Express |
| Database | MongoDB, Mongoose |
| Auth | JWT + token blacklist |
| AI | Google Gemini API |
| Deployment | Vercel |

## Getting Started

### Prerequisites

- Node.js >= 18.x
- MongoDB (local or Atlas)
- Google Gemini API key

### Backend Setup

```bash
cd Backend
npm install
```

Create a `.env` file in `Backend/`:

```env
PORT=8000
MONGO_URL=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_api_key
```

Run the server:

```bash
npm run dev
```

### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

App runs at `http://localhost:5173`.

## Environment Variables

| Variable | Description |
|---|---|
| `PORT` | Backend server port |
| `MONGO_URL` | MongoDB connection string |
| `JWT_SECRET` | Secret for signing JWTs |
| `GEMINI_API_KEY` | Gemini API key for AI features |
